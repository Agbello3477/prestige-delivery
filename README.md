# Prestige Delivery & Logistics Services

A comprehensive logistics platform featuring a Mobile App (Riders/Customers), an Admin Dashboard, and a Node.js Backend.

## Project Structure

- `/backend`: Node.js, Express, Prisma (PostgreSQL), Socket.io.
- `/admin-dashboard`: React, Vite, Tailwind CSS, Firebase.
- `/mobile-app`: React Native, Expo, NativeWind, OSM Maps.

## Live Environments

- **API Backend**: https://prestige-delivery-backend.onrender.com
- **Web Dashboard**: https://prestige-delivery-89e31.web.app

## Development

### Backend
1. `cd backend`
2. `npm install`
3. `npm run dev`

### Admin Dashboard
1. `cd admin-dashboard`
2. `npm install`
3. `npm run dev`

### Mobile App
1. `cd mobile-app`
2. `npm install`
3. `npx expo start`
